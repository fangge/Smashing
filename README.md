# Smashing
Smashing front-end project 

Smashing 网站前端文件，grunt构建

## 文件结构

* trunk - 构建包
  * css - 样式
  * img - 图片
    * sprite - 雪碧图：<a href="http://www.cn.spritegen.website-performance.org/" target="_blank">雪碧图生成器</a>
  * js - JS文件
  * forget.html 忘记密码
  * login.html 登陆
  * reg.html 注册
  * ny.html 文章页
  * index.html 首页
* psd 设计稿
